# Поддержка Selectel Cloud

Начиная с версии проекта, добавлена полная поддержка Selectel Cloud наравне с Yandex Cloud.

## Настройка Selectel

### 1. Создание сервисного пользователя

1. Войдите в панель управления Selectel (my.selectel.ru)
2. Перейдите в раздел **Аккаунт** → **Пользователи** → **Сервисные пользователи**
3. Создайте нового сервисного пользователя с необходимыми правами
4. Сохраните имя пользователя и пароль

### 2. Создание файла с учетными данными

Создайте файл `/etc/yandex-auto-up/selectel-credentials.json`:

```json
{
  "username": "your_service_user_name",
  "password": "your_service_user_password",
  "account_id": "123456",
  "project_id": "your_project_id"
}
```

Где:
- `username` — имя сервисного пользователя
- `password` — пароль сервисного пользователя
- `account_id` — номер вашего аккаунта (видно в правом верхнем углу панели управления)
- `project_id` — ID проекта облачной платформы (опционально, можно выбрать при импорте)

### 3. Импорт серверов

В CLI-панели:
1. Выберите `Profiles` → `Import from Selectel`
2. Выберите проект
3. Выберите серверы для мониторинга
4. Укажите IP-адреса для health-check

### 4. Ручное создание профиля

При создании профиля вручную:
1. Укажите `provider: selectel`
2. Укажите `project_id` — ID проекта
3. Укажите `instance_id` — ID сервера
4. Укажите `check_host` — IP-адрес для проверки доступности

## Статусы серверов Selectel

Daemon понимает следующие статусы серверов Selectel:
- `ACTIVE` — сервер работает
- `SHUTOFF` — сервер остановлен (будет автоматически запущен)
- `BUILD` — сервер создается
- `REBOOT` — сервер перезагружается

## Различия с Yandex Cloud

- Для Yandex Cloud используются Service Account ключи в `/etc/yandex-auto-up/keys/`
- Для Selectel используется файл с учетными данными `/etc/yandex-auto-up/selectel-credentials.json`
- Можно одновременно мониторить серверы в обоих облаках
- Каждый профиль имеет поле `provider` со значением `yandex` или `selectel`

## API Selectel

Проект использует:
- Identity API для аутентификации (получение IAM-токенов)
- Compute API для управления серверами
- Projects API для получения списка проектов

Документация: https://docs.selectel.ru/
